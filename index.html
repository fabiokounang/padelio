What is Padelio?

Americano is a fun tournament format where you keep playing short matches while rotating partners and opponents. Your goal is simple: score as many points as you can across all rounds. This app helps you set up a tournament, generate matches per court, track scores fast, and view a live leaderboard.

How scoring works

Each match is played to a target score (16 / 21 / 24 / 32). If Team A scores X, Team B automatically becomes (target - X). This keeps matches consistent and fast, perfect for social tournaments.

Best use cases

Weekend gatherings, club sessions, friendly tournaments, and events where you want everyone to play equally. Add players, choose courts, start, and let the app handle the round flow.

About & FAQ

This app is designed for quick Americano sessions: create a tournament, rotate players, track scores, and keep the vibe fun.

How do points work in this app?

You pick a target score per match (16 / 21 / 24 / 32). When you input Team 1’s score, Team 2 is automatically set to (target - score). This prevents long matches and keeps the tournament moving smoothly.

Does it save my data?

Yes. Your tournaments and scores are stored only in your browser (eg. Safari, Chrome, Firefox, etc) so you can reopen the app without losing progress.

Is this official?

This is a community utility app. It’s made to support friendly games and club sessions, not as an official federation tool.

How to install the app (PWA)

iPhone / iPad (Safari):
1) Open the website in Safari (not Chrome).
2) Tap the Share button (square with an arrow).
3) Scroll and tap Add to Home Screen or click More and find Add to Home Screen.
4) Rename it if you want, then tap Add.

Android (Chrome):
1) Open the website in Chrome.
2) If you see Install app, tap it and confirm.
3) If you don’t see it, tap ⋮ (top-right) → Install app or Add to Home screen.
4) Tap Install / Add to finish.
